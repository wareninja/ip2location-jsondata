IP2Location-jsondata

Generate JSON data from IP2Location Database (DBx.csv) files

A simple & readable Java code, reusable to generate JSON data files from [IP2Location Database (DBx.csv)](http://lite.ip2location.com) files! This library also makes an easy to use executable JAR, so you don't need to deal with the codebase.

How to use this library?

Option.1: use only the executable JAR!
Just download the latest jar from [releases](https://github.com/wareninja/ip2location-jsondata/releases)
Then you can run that jar file/lib from the command line. For example; 


$ java -jar ip2location-jsondata-0.0.1.jar data-folder-is-here/ IP2LOCATION-LITE-DB11.CSV

Remember that, you must have JVM/JDK in your local environment to be able to run this jar!

Option.2: use the source code!
Clone the project source code from github, then import as maven project in your ide!
Then; 

$ cd your_repo_root/repo_name
$ mvn clean install
this will generate the snapshot of executable jar. You can run this JAR file as described in the prev option above!
whenever in doubt, read the source luke!

What about the generated JSON data files?

Each generated file contains JSON representation of each IP record per line. Example record per line; 



{"country_code":"JP","country_name":"Japan","region_name":"Tokyo","city_name":"Tokyo","zip_code":"214-0021","time_zone":"+09:00","_id":16797696,"ip_to":16798719,"ipaddress_from":"1.0.80.0","ipaddress_to":"1.0.83.255","_created":1424270870645,"createdAt":"2015-02-18T14:47:50+0000","lat":35.689506,"lng":139.6917,"latlng":[35.689506,139.6917]}

These JSON data can be used for variety of purposes.
The most useful of these is to store them to MongoDB, then leverage the power of MongoDB search from your applications!
And you can easily import these JSON files into MongoDB with [mongoimport](http://docs.mongodb.org/manual/reference/program/mongoimport)
For example; 

$ cd your_repo_root/repo_name
$ mvn clean install
