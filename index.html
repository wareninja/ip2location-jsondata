Welcome to GitHub Pages.

A simple & readable Java code, reusable to generate JSON data files from IP2Location Database (DBx.csv) files! This library also makes an easy to use executable JAR, so you don't need to deal with the codebase.

How to use this library?

Option.1: use only the executable JAR!
Just download the latest jar from [releases](https://github.com/wareninja/ip2location-jsondata/releases)
Then you can run that jar file/lib from the command line. For example; 


$ java -jar ip2location-jsondata-0.0.1-jar-with-dependencies.jar data-folder-is-here/ IP2LOCATION-LITE-DB11.CSV

Remember that, you must have JVM/JDK in your local environment to be able to run this jar!

Option.2: use the source code!
Clone the project source code from github, then import as maven project in your ide!
Then; 

$ cd your_repo_root/repo_name
$ mvn clean install
this will generate the snapshot of executable jar. You can run this JAR file as described in the prev option above!
whenever in doubt, read the source luke!
